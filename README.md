# APEX_MAP_ROTATION_BOT

APEX のランクマッチのマップのローテーションが分かる bot です

## 使用方法

bot をメンションするだけで、以下の内容が分かります

- 現在のランクマッチのマップ
- 次のランクマッチのマップ
- マップが切り替わるまでの時間

## 使用技術

- discord.js v14
- TypeScript
- biome
- zeabur
